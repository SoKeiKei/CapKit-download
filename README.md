## ✨ 核心特性

- **视频实时预览**：鼠标悬停即可触发极速视频预览，告别空看标题。
- **重复模板查杀**：智能识别冗余下载，一键清理释放 G 级磁盘空间。
- **置置顶安装**：模板一键导入并置顶，无需在剪映里翻找。
- **单文件免安装**：只有 11MB，不留后台，不占资源，安全纯净。
- **本地归类管理**：扫描多级目录，自定义分类别名，让灵感井然有序。
