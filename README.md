## 📖 简介

CapKit 是一款专为剪映专业版用户设计的**模板管理小助手**。

作者本身是剪辑佬，深知本地模板文件夹堆积如山的烦恼，因此开发了这个仅 11MB 大小的单文件工具，帮你快速理清素材，一键同步草稿。它不是什么沉重的专业大作，而是一个随手即用的效率补丁。

## ✨ 核心特性

- **📂 本地目录管理**：自定义添加模板仓库目录。
- - **� 一键导入模板**：选中模板可一键同步至剪映，实时生效，无需重启剪映。
- **⚡ 11MB 极轻量**：单文件 EXE，绿色免安装，即点即用。
- **🧹 重复模板查杀**：识别相同模板，帮您清理冗余，节省宝贵磁盘空间。

## 🚀 快速开始

1. **下载**：获取 `CapKit 0.1.*.exe` 可执行文件。
2. **运行**：双击即可启动（绿色免安装）。
3. **设置**：点击右上角“设置”图标，添加您的模板存储位置。
4. **使用**：选中模板后点击“一键安装”即可马上同步到剪映草稿。

## � 常见问题

**Q: 为什么提示没有模板？**  
A: 请确保在设置中正确添加了包含剪映模板（通常包含 `draft_content.json`）的父文件夹。
